Para fazer um elemento "flutuar" para esquerda ou diretia, usa-se float. Float age de forma similar a {position: absolute}. Quando um elemento chega no destino com float, os elementos em sequencia tomam seu lugar. Para evitar esse comportamento, usa-se clear no próximo elemento apontando o lado no qual o elemento seguinte.

O elemento vermelho partiu para a direita com float e deixou um espaço em vazio com clear

O elemento azul partiu para a direita com float como o próximo elemento não possui {clear:right} o elemento
amarelo tomou seu lugar
